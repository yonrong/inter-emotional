# 情感密度——宇宙閃爍的表情
## *Affective Density: The Universe's Flickering Expression*
### 操作說明 Operation Manual

---

## 一、作品簡介

《情感密度》是一件以 WebGL2 粒子場與即時音訊合成為核心的互動數位藝術作品。螢幕上充滿數以百萬計的微粒子，每一顆都獨立呼吸、漂移，並對你的存在做出反應。作品提供五種情緒狀態，各自擁有專屬的視覺色彩、粒子物理行為，以及一組以漢斯·季默《星際效應》配樂為靈感即時合成的音景。

游標化身為持續旋轉的**黑洞漩渦**——你可以主動攪動粒子場；若你靜靜離開，黑洞會自己在宇宙中漫遊。

作品的核心問題是：**情感能否被視覺化為密度？渾沌中能否浮現出面孔？時間能否被聽見正在崩解？**

## 二、開啟方式

### 系統需求

| 項目 | 需求 |
|------|------|
| 瀏覽器 | Chrome 90+ / Firefox 88+ / Safari 15+ / Edge 90+ |
| 技術支援 | WebGL2、Web Audio API |
| 裝置 | 桌機、筆電、平板、智慧型手機（觸控支援） |
| 網路 | **不需要**——所有資源皆內嵌於單一 HTML 檔案 |

### 啟動步驟

1. 找到檔案 `index.html`（位於 `affective-density/` 目錄）
2. 以瀏覽器直接開啟（雙擊，或拖入瀏覽器視窗）
3. 頁面載入後，黑色背景上出現五個按鈕、操作提示與左下角狀態列
4. **點擊頁面任意位置**以啟動音訊引擎（瀏覽器安全政策要求使用者主動觸發）
5. 音訊提示淡出，作品正式開始

> **注意：** 若音訊未啟動，視覺粒子場仍會正常運作，僅缺少聲音。

## 四、完整操作方式

### 4.1 情緒切換

#### 左鍵點擊按鈕（主動選擇）
點擊畫面底部的任一情緒按鈕，直接跳至該情緒狀態。
- 視覺平滑過渡約 **2.2 秒**
- 音訊交叉淡化約 **1.2 秒**
- 同時重置自動切換倒數計時器

#### 右鍵點擊畫面（循環切換）
在畫面任意位置按下**滑鼠右鍵**（或長按觸控裝置），情緒依序循環：
```
CALM → TURBULENT → MELANCHOLIC → JOYFUL → FRAGMENTED → CALM → …
```
- 不會出現系統右鍵選單（已攔截）
- 同時重置自動切換倒數計時器

#### 自動隨機切換（被動）
若連續 **15 秒**未使用左鍵點擊按鈕或右鍵點擊畫面，系統將自動隨機切換到一個與當前不同的情緒。
- 切換後重置計時器，再等 15 秒再次切換
- 左下角 `AUTO · Xs` 即時顯示倒數秒數

### 4.2 黑洞漩渦游標

游標在進入畫面後化身為一個**持續旋轉的漩渦黑洞**：

- **外觀**：三條螺旋臂，從外緣的亮藍白色漸層至中心的深黑奇異點，約每 3.5 秒旋轉一圈
- **排斥力**：游標周圍 **45px** 範圍內，粒子受到強力排斥（推力 55px），形成一片粒子真空
- **漩渦吸積**：被排斥的粒子在約 **52px** 外圍堆積，形成藍白色吸積光環
- **軌道漩渦**：排斥同時帶有順時針旋轉分量，粒子圍繞游標螺旋飛散

> 移開游標，粒子會緩慢回流，恢復自然漂移。

### 4.3 黑洞自動漫遊（無人操作時）

若連續 **15 秒**未移動滑鼠，黑洞游標將自動在畫面上**緩慢漂移**：

- 每隔 2.5–6 秒隨機選取一個新目標位置（避開畫面邊緣 12%）
- 以約 4–5 秒平滑趨近目標，形成懶散、有機的飄移感
- 黑洞仍持續對粒子場施加排斥與漩渦效果
- **滑鼠一移動即立即恢復手動控制**，無縫銜接

> 這是作品「無人在場」時的自主狀態——黑洞成為宇宙中自我移動的存在。

### 4.4 靜音切換（M 鍵）

按下鍵盤 **`M`** 鍵，切換音訊靜音/開啟：

- 靜音時：左下角第三行顯示橙色 `[M] MUTED`，音量在 0.45 秒內平滑淡出
- 解除靜音：音量在 0.45 秒內平滑淡入至正常水準
- 若在音訊初始化前就按下 M，音訊啟動後將保持靜音狀態

### 4.5 自動循環：幻象顯現（Pareidolia Cycle）

無需任何操作——每 **30 秒**，粒子場中自動浮現三個幽靈形象：

| 時段 | 形象 | 顏色 |
|------|------|------|
| 0–7 秒 | 悲傷的臉（面龐、淚水、下垂嘴角） | 深紫 |
| 7–14 秒 | 喜悅的手（向上伸展的五指） | 金黃 |
| 14–21 秒 | 破碎的眼（虹膜、瞳孔、杏仁形眼瞼） | 青綠 |
| 21–30 秒 | 虛空（形象淡出，黑暗靜默） | — |

形象以最高 **52%** 不透明度疊加，刻意保持曖昧——它們是臉孔，還是你投射於混沌中的幻想？

## 五、五種情緒狀態詳解

### CALM — 靜謐
**視覺：** 邊緣帶綠意的深藍/深紫色調，粒子幾乎靜止，緩慢浮動  
**粒子能量：** 極低（drift = 0.28）  
**音景：** D 小調，宇宙性的空曠感
- 低音持續音：D–A 開放五度（73–220 Hz）
- 旋律：五音下行句（D4→C4→Bb3→A3→G3），每音持續 8–11 秒
- 如同孤星在虛空中緩慢冷卻

---

### TURBULENT — 湍流
**視覺：** 橙/紅/黃的灼熱色調，核心炙白，粒子劇烈旋轉並膨脹  
**粒子能量：** 極高（drift = 0.95），額外加入漩渦分量  
**音景：** 謝柏音調（Shepherd Tone）——永無止境的上升幻象
- 六個正弦波，分佈於 A0–A5（相差一個八度）
- 頻率持續上升，達到頂端後無縫循環：**永遠在升高，卻從未到達頂點**
- 模擬無法逃脫的引力，時間在此加速崩塌

---

### MELANCHOLIC — 憂鬱
**視覺：** 深紫至蒼灰色調，冷藍色生物發光閃爍，粒子沉緩如沉默  
**粒子能量：** 極低（drift = 0.18），動作遲緩而沉重  
**音景：** A 小調，悲慟的心跳
- 和弦：A1–E2–A2–C3–E3（厚重、無法移動的哀慟）
- 旋律：A3→G3→F3→E3→F3→G3（始終無法完整上揚）
- 心跳脈衝：約 43 BPM，「Lub」(55 Hz, 12ms 起音)＋「Dub」間隔 240ms

---

### JOYFUL — 喜悅
**視覺：** 金黃光芒從畫面上方中央輻射，整體場域明亮溫暖  
**粒子能量：** 中高（drift = 0.72），充滿節奏感的前進動能  
**音景：** D 大調，田園牧歌式的分解和弦
- 低音：D2 + A2（溫暖的大地根基）
- 12 音分解和弦，以 6/8 拍感約 112 BPM 流動
- **唯一具有向前律動感的狀態**——時間在此重新流動

---

### FRAGMENTED — 碎裂
**視覺：** 毒綠/深紅/虛空灰，水平掃描線（glitch bands）橫切畫面  
**粒子能量：** 最高混沌（drift = 1.0）  
**音景：** 機械的節拍計時
- 管風琴基底：C 大調（C2–E3，冷漠而超然）
- 機械滴答：每 **1.246 秒**一次（對應《星際效應》米勒星球的地球時間：每滴答 = 17 小時）
- 880 Hz + 1320 Hz 方波，帶通濾波，65ms 衰退
- 時間不再是體驗，而是一種無情的計量

## 七、黑洞游標視覺架構

游標以 Canvas 2D 即時繪製，每幀更新（60fps），由三層疊加構成：

```
        外圍環形霧光（淡藍紫，R=24px）
            ↓
  三條螺旋臂（外亮藍白 → 內透明）
     ‧ 每條旋轉 1.5 圈
     ‧ 約每 3.5 秒完整旋轉一週
            ↓
      奇異點黑心（R≈13px，深黑漸層）
```

**粒子場中的黑洞效應（shader 層）：**

| 層次 | 半徑 | 效果 |
|------|------|------|
| 排斥推力 | 0–45 px | 強制彈開粒子（力 55px），形成真空 |
| 軌道漩渦 | 9–45 px | 粒子圍繞游標順時針旋轉飛散 |
| 吸收陰影 | 0–28 px | 粒子顏色加深 88%（深黑感） |
| 吸積光環 | ~52 px | 藍白色亮環（堆積粒子的發光邊界）|

## 八、音訊架構說明

所有聲音為**即時合成**，無任何外部音訊檔案：

```
AudioContext
    │
    ├── 5 個情緒音景（各自獨立的 gain bus）
    │       ├── 乾訊號（dry）
    │       └── 濕訊號（wet）─── 共享混響
    │
    ├── 大教堂混響（Convolution Reverb）
    │       └── 3.5 秒指數衰退白噪音脈衝，42% 混入比例
    │
    └── 主輸出增益（Master Gain）
            ├── 啟動時 2.5 秒淡入至 0.15 音量
            └── M 鍵：0.45 秒平滑淡出 / 淡入
```

音景切換時，兩個狀態交叉淡化（1.2 秒）避免突兀截斷。

## 九、常見問題

**Q：畫面是黑色的，什麼都看不到？**  
A：確認瀏覽器支援 WebGL2，可至 `get.webgl.org` 驗證，或嘗試更新瀏覽器。

**Q：沒有聲音？**  
A：必須先點擊頁面任意位置以啟動 AudioContext。若已點擊仍無聲，確認 M 鍵是否誤觸靜音（左下角顯示橙色 MUTED），再檢查系統音量。

**Q：iOS 裝置上音訊無法啟動？**  
A：作品已針對 iOS 加入 `touchend` 事件觸發與 `AudioContext.resume()`，請使用 Safari 15+ 或 iOS 上的 Chrome。

**Q：黑洞自己在移動？**  
A：若滑鼠靜止超過 15 秒，黑洞會進入自動漫遊模式，這是正常行為。移動滑鼠即可立即恢復手動控制。

**Q：右鍵沒反應？**  
A：需點擊在畫面的 `<canvas>` 範圍內（全螢幕）。若在 UI 按鈕上按右鍵，行為由瀏覽器決定。

**Q：幻象形象為什麼若隱若現？**  
A：刻意設計——最高不透明度僅 52%，保留曖昧性，讓觀者的大腦完成剩餘的感知工作。

**Q：效能卡頓？**  
A：GPU 資源不足時縮小瀏覽器視窗，或關閉其他 GPU 密集型分頁。

## 十一、致謝與脈絡

本作品的聲音設計靈感源自 **Hans Zimmer** 為《星際效應》（*Interstellar*, 2014）所創作的配樂，五種情緒狀態分別對應：

| 情緒 | 對應曲目 | 音樂特徵 |
|------|----------|----------|
| CALM | *Day One* | D 小調，宇宙性的空曠 |
| TURBULENT | *No Time for Caution* | 謝柏音調，永無止境的上升 |
| MELANCHOLIC | *Stay* | A 小調，心跳脈衝，哀慟 |
| JOYFUL | *Cornfield Chase* | D 大調，田園式分解和弦 |
| FRAGMENTED | *Mountains* | 機械節拍，每秒 = 17 地球小時 |

視覺語言援引了宇宙學中**密度波理論**的意象——星系的螺旋臂並非實體物質的結構，而是密度的漣漪在恆星間傳播。情感或許也是如此：不是固態的事物，而是一種傳遞中的疏密變化。

而那個持續旋轉、在宇宙中孤獨漫遊的黑洞，是作品中唯一真正屬於「你」的東西。
