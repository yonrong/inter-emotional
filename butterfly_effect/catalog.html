Inter-Emotional
情感交錯

Inter-Emotional（情感交錯）是一場探索情感如何在人與介面之間流動、交會、相互塑造的畢業創作展。展覽名稱中的「Inter」，意指「之間」——情感不存在於人的內部，也不存在於介面的表面，而是在兩者的接觸地帶持續生成。

本展包含三件數位互動裝置：《蝴蝶效應》（Butterfly Effect）、《夜之流 I–IX》（Fluss der Nacht I–IX）與《情感密度——宇宙閃爍的表情》（Affective Density: The Universe's Flickering Expression）。三件作品分別以混沌動力學、時間記憶與粒子密度三個切入點，共同探索同一個問題：介面，能不能感受？

論文框架 · Thesis Framework

本展與研究論文《從感知到微互動：數位閱讀平台情感化介面設計研究》（From Perception to Micro-Interaction: Affective Interface Design for Digital Reading Platforms）相互呼應。論文提出：在數位介面設計中，感知（Perception）是情感體驗的入口；微互動（Micro-Interaction）是感知與情感之間的橋梁；情感化介面設計（Affective Interface Design）則是讓兩者在使用者經驗中無縫整合的設計實踐。

《蝴蝶效應》將這套框架從文字研究帶入可感受的空間，作為論文核心概念的藝術實踐與視覺論證。

Chapter I
Artist Statement
藝術家陳述

「一隻蝴蝶在巴西扇動翅膀，是否會在德克薩斯州引發一場龍捲風？」
"Does the flap of a butterfly's wings in Brazil set off a tornado in Texas?"

— Edward N. Lorenz, 1972

我開始思考介面設計中的「感知」問題，是從一次很微小的失敗開始的。

我正在設計一個數位閱讀平台，試圖讓使用者在滑動頁面時感受到某種「閱讀的重量」——那種翻動實體書頁時，紙張在手指間的輕微阻力。我試過各種動態曲線、各種回彈係數。結果都差一點。那個「差一點」讓我困惑很久：技術上正確的互動，為什麼不一定能引發正確的感受？

後來我讀到了 Lorenz 的故事。1963 年，他在重新輸入氣象模擬數據時，將 0.506127 四捨五入為 0.506。差距只有 0.000127——遠小於任何可感知的誤差。但兩個月後，模擬的天氣系統已經完全不同。不是稍有偏差，而是另一個世界。

我突然明白了那個「差一點」是什麼。感知不是對刺激的被動接收，而是一種主動的、對初始條件極度敏感的詮釋過程。一個微小的觸覺細節——一毫秒的回應延遲，一個像素的位移差異——可能決定使用者是否感受到「這個介面懂我」。就像那 0.000127，決定了一整個天氣系統的命運。

《蝴蝶效應》是我試圖讓這個認識變得可見的嘗試。六十萬個光點，在你眼前形成兩隻蝴蝶。當你的手指觸碰螢幕，吸引子被打破，粒子逸出軌道。你的一個輕觸，改變了六十萬個粒子的命運。

這就是微互動的本質：一個小的輸入，一個大的情感輸出。而兩者之間的那條路徑，就是介面設計所在的地方。

Chapter II
Conceptual Framework
概念框架

感知的入口 · Perception as Entry Point

感知先於認知，先於情感，先於任何有意識的反應。當觀者站在這件作品前，眼睛首先接收到的是光點的密度、顏色的溫度、運動的速度感。這些前語言的感知訊號，在使用者意識到「這是一個 Lorenz 吸引子」之前，就已經開始塑造情感基調。

作品的視覺設計因此從感知層次出發：深紫黑的背景對應沉浸感；加法混色的發光粒子對應自發光、有生命力的質感；兩翼對稱的色相差異對應「同源而殊途」的視覺敘事。這些都是感知層次的情感設計，不需要任何說明文字，就能在觀者的神經系統中直接作用。

微互動作為詩意 · Micro-Interaction as Poetic Gesture

微互動是介面設計中最小的有意義互動單位：一個點擊、一個滑動、一個懸停。在功能性介面設計中，微互動的目標是「讓使用者知道發生了什麼」。但在這件作品中，微互動承擔了更深的任務：讓使用者感受到自己的存在改變了世界的狀態。

蝴蝶效應，就是微互動的詩意版本——一個小的輸入，觸發一個不可逆的、蔓延的變化。你的觸碰讓六十萬個粒子進入了它們原本不會進入的軌道，而那些軌道，再也回不去了。

情感化介面的三個層次 · Three Levels of Affective Interface

依據 Donald Norman 的情感設計框架，作品嘗試在三個層次同時運作：

本能層（Visceral）：視覺上的發光粒子群、深邃的黑色空間、音樂同步的色彩，在觀者還未理解任何概念之前，就觸發了「美麗」的直覺反應。

行為層（Behavioral）：觸碰的回應是即時的、物理感的——漣漪從游標向外傳播，粒子被推開或被吸引。每一個動作都有清晰可感的後果，建立了觀者與作品之間的對話感。

反思層（Reflective）：當觀者意識到「我的一個輕觸，改變了六十萬個粒子的命運」，這個反思時刻才是作品真正想要抵達的地方——一種關於自身存在的感受：我是這個系統的一部分，我的行動有重量。

Chapter III
Work Description
作品解說

《蝴蝶效應》以六種狀態持續循環演化，每個狀態對應一種情感質地與運動邏輯：

Mode 0
Dual Wings in Flight
比翼雙飛

兩條 Lorenz 軌道從微小的初始差異出發，預熱四千步後完全去相關。六十萬個光點分屬兩翼，在三維空間中緩緩自轉。這是作品的「靜觀」狀態：感知系統運行中，等待介入。

情感質地：沉靜、等待、內斂的張力

Mode 1
Ephemeral Drift
Drift · 漂流

Lorenz 束縛解除，粒子進入三維噪聲場漫遊。色彩隨位置緩緩變化，像墨水在水中暈散。這個狀態模擬情感感知中的「漫遊」——沒有目標的注意力，往往最能吸收細節。

情感質地：放鬆、溶解、無目的的開放

Mode 2
Orbital Vortex
Vortex · 漩渦

每一翼繞著各自的鞍點公轉，形成雙螺旋旋轉場。呼應情感體驗中的「糾纏感」——被一個念頭反覆拉回，卻又無法停在同一個點上。

情感質地：糾纏、旋轉、無法掙脫的拉力

Mode 3
Breathing Pulse
Pulse · 脈衝

週期性徑向力從鞍點膨脹再收縮，頻率約 0.25 Hz，接近人類靜息心率的一半。粒子群以近乎生物的節律呼吸，呼應感知研究中的「節律同步」現象。

情感質地：生命感、節律、共振

Mode 4
Lissajous Gravity
Gravity Well · 引力井

三個引力點沿 Lissajous 曲線移動，各自以不同頻率震盪。粒子在競爭引力場中形成複雜流線，隱喻情感決策中的「多重吸引」——當一個人同時被幾種情感拉扯，軌跡的複雜性就是那份內在張力的視覺化。

情感質地：拉扯、張力、多重吸引的複雜性

Mode 5
Wave Field
Fleeting Glimpses · 轉瞬浮光

三個方向的正弦波力場同時作用，形成互相干涉的三維波紋。粒子在看似規則的場中展現不可預測的軌跡——對應情感感知中的「一瞥」，那些稍縱即逝、難以言說的情感印象，像光打在水面上的瞬間折射。

情感質地：短暫、閃光、無法捕捉的美麗

Chapter IV
Perceptual Design
感知設計說明

色彩與情感感知

作品使用加法混色（Additive Blending）在深黑背景上疊加自發光粒子。這個技術選擇模擬了暗室中的光源行為，觸發感知系統中對「自發光物體」的特殊反應——自然界中會自己發光的東西，通常是活的（螢火蟲、深海生物）或重要的（火、星星）。粒子因此獲得一種非語言的「生命感」。

色彩系統以 HSV 色彩空間運算，以色相（Hue）週期性映射位置與速度。兩翼之間有固定的 16% 色相偏移，在視覺上區分身份，卻共享同一套亮度規則——同源而殊途的感知設計。

音景與情感同步

每個 Mode 搭配一首音樂，遵循「情感色調一致性」原則：視覺運動的節奏感、密度感、方向感，與音樂的速度、音色、動態弧線相互呼應。模式切換時，2.5 秒的交叉淡變讓情感過渡有緩衝，避免突然的情感斷裂。

Chapter VII · 情感密度
Affective Density
作品概念與解說

一個覆蓋全螢幕的微粒子宇宙——每隔 5 像素一顆，數以萬計的光點在深黑的空間中緩緩呼吸。它們的顏色、密度、運動速度，由一個隱藏的情感狀態所驅動。當你切換情感，整個宇宙都在改變。

《情感密度》探索的核心問題是：情感能不能成為一種密度？ 能不能讓人在看到粒子場的第一秒，不需要任何文字說明，就感受到「這是平靜」或「這是激盪」？五種具名的情感狀態——不是隱喻，而是直接的情感語言——等待觀者選擇進入。

論文連結 · Thesis Connection

感知（Perception）：粒子場以前語言的視覺密度和色彩傳達情感——觀者在理解任何標籤之前，已經「感受到」情感狀態的差異。這正是感知先於認知的設計實踐。

微互動（Micro-Interaction）：鼠標移動在 90px 範圍內排斥粒子；情感按鈕切換觸發 0.45 秒的全場過渡。每一個細微動作，都在整個粒子宇宙中留下漣漪。

情感化介面（Affective Interface）：五個具名的情感狀態是介面直接對話情感語言的嘗試——不是功能按鈕，而是情感的選擇器。

五種情感狀態 · Five Emotional States

State 0
Calm
靜謐

深邃黑暗中，藍綠邊緣向深藍紫核心漸層，生物螢光的閃光如同深海微光點點。粒子緩慢呼吸，場的密度均勻而安定。

音景：D 小調管風琴低音 + 宇宙共鳴（Hans Zimmer《Day One》式）

情感質地：靜謐、沉浸、充滿的寂靜

State 1
Turbulent
激盪

橙紅黃的粒子場翻騰，白熱核心向外噴射，粒子在暴力的抖動中旋轉。Risset 無限上升音調持續堆疊——一個永遠無法抵達頂點的焦慮。

音景：Risset 無限上升音 + C 小調音床（Hans Zimmer《No Time for Caution》式）

情感質地：焦慮、緊張、無法停止的升騰

State 2
Melancholic
憂鬱

深紫到退縮的灰藍，冷色調的生物螢光閃爍如同雨中城市燈火。43 BPM 的心跳聲緩緩跳動——比靜息更慢，這是悲傷的心跳頻率。

音景：Am 管風琴 + 43BPM 心跳 + 下行旋律 A3→E3→G3（Hans Zimmer《Stay》式）

情感質地：憂傷、退縮、緩慢的心跳

State 3
Joyful
喜悅

金橙的粒子從上方中央輻射而出，溫暖光芒佈滿整個場。D 大調琶音在約 112 BPM 的節奏中向上爬升。那個向上伸展的手，就在光的中央等待你的目光。

音景：D 大調琶音 + ~112BPM + D2 持續低音（Hans Zimmer《Cornfield Chase》式）

情感質地：溫暖、上升、開放的喜悅

State 4
Fragmented
碎裂

毒綠、深猩紅、虛空的灰交織，水平故障帶切割整個場。固定在 1.246 秒的節拍計時聲是場景中唯一的規律——碎裂，但有節奏。

音景：C 大調管風琴 + 每 1.246 秒節拍計時（Hans Zimmer《Mountains》式）

情感質地：混亂、碎裂、但有節律的秩序

Chapter VIII · 情感密度
Pareidolia & Interaction
幽靈識認與互動設計

幽靈識認 · The Ghost Shapes

在粒子場的光影之間，三個幽靈形態在 30 秒的循環中出現又消散。它們從未以固體出現，只是密度的聚集，讓觀者的感知系統在噪聲中識別出熟悉的形狀。

這是「幽靈識認」（Pareidolia）現象的設計化運用：人腦天生傾向在視覺模式中尋找臉、手、眼睛。在這件作品中，這種認知偏見成為情感體驗的媒介。你在粒子場中「看見」的那張臉，是作品放進去的，還是你的感知系統創造出來的？

幽靈形態色調細節
憂傷之臉 · Sad Face深紫橢圓臉輪廓、兩個眼孔、向下弧線嘴角、一滴眼淚
喜悅之手 · Joyful Hand金橙掌心、手腕、五指向上伸展，如同觸碰遠方的什麼
碎裂之眼 · Fragmented Eye青藍杏仁眼瞼邊緣、虹膜環、瞳孔、被噪聲切割的碎裂感

音景設計 · Soundscape Design

五種情感狀態各有獨立的程序生成音景，以 Web Audio API 即時合成，無任何預錄音頻。所有音效通過 3.5 秒大教堂殘響（Convolution Reverb）處理，在視覺的微粒子宇宙中創造出聲學的空間感。

音景的靈感來源是 Hans Zimmer 為《星際效應》（Interstellar）所創作的管風琴聲響美學——以教堂管風琴的泛音結構，讓情感在宇宙的尺度中共鳴。

感謝所有為本展提供背景音景的音樂家，他們的聲音讓數學擁有了溫度，讓六十萬個粒子不只是粒子，而是有呼吸的存在。

感謝每一位停下腳步，願意用一根手指觸碰光點的觀者。你的那一觸，就是那隻扇動翅膀的蝴蝶。在那一刻，你和這件作品之間，有了情感交錯。

這件作品以網頁瀏覽器即時運算呈現，無預先渲染的畫面。
This work is rendered in real-time. No frames are pre-rendered.

每一次觀看，都是獨一無二的混沌事件。
Every viewing is a unique chaotic event.
